# Threat Intelligence and Threat Hunting Using MITRE ATT&CK, SOCradar and MITRE ATT&CK Navigator


---
## Project overview
This project analyzes threat actor activity targeting the Information Services through structured intelligence collection and MITRE-based mapping. Using SOCradar and the MITRE ATT&CK, three advanced threat groups were profiled and their documented tactics, techniques, and procedures (TTPs) were mapped and visualized in MITRE ATT&CK Navigator. A combined overlap layer was generated to identify shared behavioral patterns across actors. The findings highlight high-priority detection opportunities and reinforce the importance of behavior-based defensive strategies within the sector.


---

### Threat Actors Analyzed

The following threat actors targeting the Information Services were selected using SOCradar to and analyzed:

- Volatile Cedar
- APT32
- APT41

Each actor was evaluated based on vailability of mapped tactics and techniques within the MITRE ATT&CK.

--- 

### MITRE ATT&CK Mapping & Overlap Analysis

For each threat actor:

- Individual layers were created using MITRE ATT&CK Navigator
- Documented tactics, techniques, and procedures (TTPs) were mapped
- Actor-specific Excel sheets were generated
- A combined ATT&CK Navigator layer was then generated to visualize the overlap across all three actors.

This combined layer highlights:

- Shared tactics across actors
- Frequently reused techniques
- High-prevalence behaviors within the sector

The overlap analysis enables identification of:

- Detection priority techniques
- Behavioral patterns common to multiple adversaries
- Sector-specific attack trends

---   

### Overlap Insight

The combined layer demonstrates that although each actor has distinct operational characteristics, several techniques are reused across campaigns. These overlapping techniques represent critical monitoring and detection opportunities for organizations operating within the Information Services sector.
